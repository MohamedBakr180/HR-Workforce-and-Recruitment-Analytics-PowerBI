# Screenshots

This folder contains visual dashboards and analytics from the HR Workforce and Recruitment Analytics PowerBI project.

## Contents

### 1. **workforce_overview.png**
Overview of the workforce metrics and key performance indicators including headcount, departments, and employee distribution.

### 2. **recruitment_analysis.png**
Detailed recruitment analytics showing hiring trends, recruitment pipeline, and candidate flow analysis.

### 3. **engagement_satisfaction.png**
Employee engagement and satisfaction metrics with survey results and sentiment analysis.

### 4. **training_effectiveness.png**
Training program effectiveness metrics including completion rates, skill development, and training ROI.

**How to view these dashboards:**
- These screenshots represent key pages from the PowerBI dashboard
